# Launch Plan (Week 1)

## Day 0 — Soft Launch
- Publish v0.1.0 with clear scope and contribution guidelines.
- Open starter issues: add-entry, dead-link, taxonomy-improvement.

## Day 1 — Submit to Awesome lists
- Submit to relevant awesome repos (LLM, agents, memory, RAG).
- Keep the pitch factual: taxonomy + starter paths + maintenance automation.

## Day 2 — Reddit/HN
### Reddit template
Title: Curated 60+ LLM memory resources with taxonomy and starter paths
Body: Why this exists, what it includes, limitations, and invitation for corrections.

### HN template
Title: Show HN: Awesome LLM Memory (60+ entries, linted links)
Body: Curation principles, quickstart, request for edge cases.

## Day 3 — Twitter/X thread
1) Why memory resources are fragmented
2) Taxonomy and starter paths
3) 5 must-read papers
4) 3 repos to try this weekend
5) Invite PRs

## Day 4-7 — Maintenance cadence
- Run link checks every 48 hours.
- Merge low-risk PRs quickly.
- Publish weekly mini changelog.
