# Taxonomy

This repo organizes LLM memory resources by practical use:

1. Surveys & Taxonomies
2. Systems & Frameworks
3. Memory Representations
   - summary/notes
   - vector/retrieval memory
   - graph/knowledge memory
   - hybrid/hierarchical
4. Writing / Updating / Forgetting
5. Benchmarks & Evaluation
6. Datasets
7. Related Awesome Lists

Memory scope tags:
- `short`: within-prompt / immediate context
- `session`: multi-turn session memory
- `user`: personalization and preference memory
- `long`: persistent long-horizon memory
